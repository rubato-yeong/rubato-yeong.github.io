# TL; DR 🚀

* **Sioyek**은 PDF 리더로, 논문 읽기에 최적화되어 있다.
* 뭔가 Adobe PDF나 Google Chrome Extension과는 다르다. 뭔가 멋있다.
* 자주 쓰는 기능 및 커스터마이징 내용은 일주일 이상 사용해보고 정리할 예정이다.

<br>

# 1. Key Features

* 파일 열기
  * `o` 키로 파일 열기
  * `Shift o` 키로 기존에 열었던 파일 선택하여 열기
  * `delete`키로 기존에 열었던 파일 목록 삭제 (`Shift o` 이후 가능)
* 페이지 이동
  * `t`로 TOC 보고 선택하여 이동
  * `space` 및 `Shift space`로 페이지 빠르게 이동
  * `backspace`로 원래 있던 위치로 이동
  * `gg`로 첫 페이지, `ngg`로 n 페이지, `G`로 마지막 페이지 이동
  * `gc`로 다음 챕터, `gC`로 이전 챕터 이동
  
그 외 작성 중입니다.
